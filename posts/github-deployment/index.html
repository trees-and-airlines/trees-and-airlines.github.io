Deploying to GitHub

GitHub is a good place to host a small blog for free.

trees-and-airlines Jan 31, 2021

This blog will need to eventually be hosted somewhere. There are many places that allow you to host a small blog for free. GitHub, Gitlab, and Netlify seem to be the go-to places, so I will begin by trying GitHub. I want to get the hang of the process involved in publishing a Hugo blog.
